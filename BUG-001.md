Bug ID     : BUG-001
Title      : Bug can be submitted with empty description field
Project    : Bug Tracker App
Reported By: Safayet
Date       : 13/03/2026

SEVERITY   : Low
PRIORITY   : Medium

ENVIRONMENT:
- Browser : Google Chrome
- OS      : Windows
- App     : Bug Tracker (index.html)

SUMMARY:
The app allows a bug to be submitted without a description.
Only the title field is validated. Description has no validation.

STEPS TO REPRODUCE:
1. Open the Bug Tracker app
2. Enter any text in the "Bug title" field
3. Leave the "Describe the bug" textarea empty
4. Select any severity from the dropdown
5. Click "Add Bug"

EXPECTED RESULT:
An alert should appear saying "Please enter a description!"
The bug should NOT be added to the list.

ACTUAL RESULT:
The bug is added to the list with an empty description.
No error or alert is shown.

ATTACHMENTS:
- Screenshot of bug card with empty description field

SUGGESTED FIX:
Add a validation check for the description field in the
addBug() function inside script.js, similar to the existing
title validation.
