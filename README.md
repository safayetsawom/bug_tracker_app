# Bug Tracker App for projects
A bug tracking web app built from scratch with HTML, CSS, and JavaScript.

# Features
- Add bugs with title, description and severity level
- Mark bugs as fixed or reopen them
- Delete bugs
- Data persists using localStorage

# Tech Stack
- HTML5
- CSS3
- JavaScript (Vanilla)

# QA Documentation
This project includes full QA documentation:
- `test-cases.md` — 9 manual test cases covering all features
- `BUG-001.md` — Jira-style bug report for a validation issue found during testing
